# DCAgent

an analytics sdk for HTML5 app and web game

## Demo

```bash
npm run demo
```

## Unit test

in Node.js

```bash
npm run test-node
```

in browser

```bash
npm run test
```

## Build

```bash
npm install
gulp
```
